# Trabajo-SED
Repositorio del trabajo de Sistemas Electrónicos Digitales, formado por dos proyectos independientes:  Control de una lavadora en VHDL sobre FPGA, basado en máquinas de estados y diseño síncrono.  Juego “Simon Dice” implementado sobre la STM32F407G-DISC1, utilizando control secuencial y gestión de E/S digitales.
